# 🎤 音声を文字に変換 - 聞き取りサポートアプリ

98 歳のおばあさんのために作った、超シンプルな音声認識アプリです。
ボタンを長く押すだけで、話した言葉が大きく画面に表示されます。

## 🎯 使い方（3 ステップ）

### 1️⃣ アプリを起動

**Windows の場合：**

- `起動.bat` をダブルクリック

**Mac/Linux の場合：**

```bash
bash 起動.sh
```

### 2️⃣ ブラウザを開く

```
http://localhost:5000
```

### 3️⃣ ボタンを長く押して話す

- **ボタンを押す** → マイク許可 → 話す
- **ボタンを離す** → 自動で文字に変換

それだけ！超シンプルです。

## 🔧 初回セットアップ（最初だけ）

### Windows PowerShell で以下を実行：

```bash
cd speech-to-text-app
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Mac/Linux で以下を実行：

```bash
cd speech-to-text-app
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

---### 4. API キーを設定

**Windows（PowerShell）の場合:**

```powershell
$env:OPENAI_API_KEY="sk-xxxxxx...ここにAPIキーを貼り付けます"
```

**Windows（コマンドプロンプト）の場合:**

```cmd
set OPENAI_API_KEY=sk-xxxxxx...ここにAPIキーを貼り付けます
```

**Mac/Linux の場合:**

```bash
export OPENAI_API_KEY="sk-xxxxxx...ここにAPIキーを貼り付けます"
```

### 5. アプリケーションを起動

```bash
python app.py
```

### 6. ブラウザでアクセス

ブラウザで以下の URL にアクセス：

```
http://localhost:5000
```

## 📖 使い方

1. **🎙️ 聞き取り開始ボタン** をクリック
2. マイクの使用を許可（ブラウザが聞く場合）
3. 周りの音声が自動で文字に変換されます
4. **⏹️ 停止ボタン** で一時停止
5. 文字サイズを好みに調整（推奨：大きい）

## ⚙️ 設定オプション

- **自動文字起こし**: チェックを入れると、連続で聞き取りを続けます
- **文字サイズ**: 大きい/普通/小さい から選択可能

## 🔒 セキュリティとプライバシー

- マイク入力は OpenAI サーバーに送信されます
- 音声は文字起こし後、OpenAI のサーバーに保存されない設定になっています
- 機密情報は扱わないようにしてください

## 🐛 トラブルシューティング

### マイクが認識されない場合

- ブラウザでマイクの使用を許可しているか確認
- ブラウザの設定 → プライバシー → マイクで許可を確認
- 別のアプリでマイクが使用中でないか確認

### 文字起こしが開始されない場合

- OpenAI API キーが正しく設定されているか確認
- インターネット接続を確認
- API の利用残高を確認（https://platform.openai.com/account/billing/overview）

### エラーメッセージが表示される場合

- ブラウザのコンソール（F12）でエラーを確認
- API キーの有効期限を確認
- サーバーログを確認：ターミナルに表示されるメッセージを確認
